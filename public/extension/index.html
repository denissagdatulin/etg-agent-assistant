ETG Agent Assistant

Рабочее расширение для команды incident. База знаний загружается из облака, поэтому правила обновляются без переустановки. Когда меняется сам код расширения, интерфейс внутри Chrome покажет, что нужна новая версия.

Первая установка

  1. Скачайте архив и распакуйте его в постоянную папку, которую не будете удалять.
  2. Откройте chrome://extensions/.
  3. Включите Developer mode.
  4. Нажмите Load unpacked и выберите распакованную папку.

Как обновляться дальше

  1. Если внутри расширения появилась плашка про обновление, скачайте новый архив по этой же странице.
  2. Распакуйте новую версию поверх той же папки или замените её содержимое.
  3. Вернитесь в chrome://extensions/ и нажмите Reload.

Что обновляется само

База знаний, профили команд и релизная информация живут на сервере. Их можно менять без переустановки расширения.

Что всё ещё требует действий пользователя

Сам код Chrome extension нельзя бесшовно переустанавливать с обычного сайта на Windows и macOS. Поэтому обновление кода остаётся ручным шагом через Reload.

Если архив не скачивается, обратитесь к Denis или откройте эту страницу ещё раз после обновления Vercel.
