# ETG Agent Assistant — Chrome Extension

Расширение для автоматизированной оценки тикетов в CRM ETG.

---

## Установка (режим разработчика)

### Шаг 1 — Скачать файлы
Нажать **Code → Download ZIP** на этой странице, распаковать архив.

### Шаг 2 — Открыть Chrome
Перейти в адресной строке: `chrome://extensions/`

### Шаг 3 — Включить режим разработчика
Переключатель **"Режим разработчика"** — вверху справа страницы.

### Шаг 4 — Загрузить расширение
Нажать **"Загрузить распакованное"** → выбрать папку с файлами
(ту, где находится файл `manifest.json`).

### Шаг 5 — Готово ✓
Иконка расширения появится в панели инструментов Chrome.

---

## Использование

- Откройте тикет в CRM (`crm.etg.team`)
- Нажмите на иконку расширения
- Расширение автоматически считает данные тикета и сформирует оценку
